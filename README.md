## Description

An Express.js API to use Sequelize to interact with a MySQL database.

## Installation

From the root dir

install the dependencies: \
`npm i`

## Usage

From the root dir

Create the database in MySQL run: \
`mysql -u root -p`

then: \
`source db/schema.sql `

To see the file: \
`npm run seed`

Start the express server: \
`node server.js`
